0.0.2 feat: asmSim has stack support now and a bx register

Changes:

- Added the Headerbar Options
- Added URL Parameter Options
- Added second Register called bx
- Preparations for stack pointer work

URL Parameter

Parameters Options Defaults to
"darkMode" true | false false
"lang" "mini@asm" "mini@asm"
"memSize" (<number>"k")|(<number>) "5k"
"regs" ["bx"]["bp"] ""

BX Register Instructions

LOADBI => LOADI for BX
LOADB => LOAD for BX
STOREB => STORE for BX
XCHG => Switches the two registers ACCUMULATOR and BX
0.0.3 feat: Added Examples and GetStack SetStack

Changes

- ID Change: "mini@asm" =now> "mini-asm"
- Added mobile mode => key: "mobile" defaults: ""; options: "legacy"|""
- Added GETSTACK => Loads a value to acc. Memory address is stackpointer - op1
- Added SETSTACK => Stores the value of acc to memory. Memory address is stackpointer - op1
